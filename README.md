# Learning Management System (LMS)

## About Project

This is a desktop-based Learning Management System developed using Core Java, Swing, AWT, JDBC, and MySQL. The application helps manage students, courses, and user information efficiently.

## Features

* User Login and Registration
* Student Management
* Course Management
* Add, Update, Delete, and View Records
* Database Connectivity using JDBC
* User-friendly GUI using Swing and AWT

## Technologies Used

* Core Java
* Swing & AWT
* JDBC
* MySQL

## Modules

* Admin Module
* Student Module
* Course Module
* User Management Module
